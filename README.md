i.	Какую ошибку вы внесли? 
- Для создания ошибки был добавлен заранее неверный тест, который однозначно бы не прошел проверку. Тест проверял equal 10 items, когда возможный максимум 5.
ii.	Какое сообщение об ошибке было в логах GitHub Actions?
- Error Message:
   Assert.Equal() Failure
Expected: 10
Actual:   5
iii.	На каком шаге и почему упал пайплайн?
- Пайплайн упал на шаге Run unit tests при запуске процесса dotnet test потому что добавленный тест не прошел проверку и программа завершилась с кодом ошибки.
iv.	Как вы ее исправили?
- Ошибка была исправлена с помощью комментирования нерабочего теста, который не проходил проверки.
